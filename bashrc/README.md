# bashrc

Modular Bash configuration files. This setup allows for a clean and organized `.bashrc` by splitting different concerns into separate files.

## Contents
- `.config/bashrc/`:
    - `00-init`: Initial shell setup and environment variables.
    - `10-aliases`: Command aliases for productivity and shortcuts.
    - `20-customization`: Shell behavior, prompt settings, and custom functions.
    - `30-autostart`: Commands and scripts to be executed upon starting an interactive session.
