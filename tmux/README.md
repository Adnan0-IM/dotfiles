# tmux

Configuration for [tmux](https://github.com/tmux/tmux), a terminal multiplexer that allows for multiple terminal sessions in a single window.

## Contents
- `.config/tmux/`:
    - `tmux.conf`: The main configuration file for session management, keybindings, and appearance.
    - `cheatsheet.md`: A reference for custom and standard tmux keybindings.
    - `scripts/`: Helper scripts for session management and automation.
    - `.tmux/`: Directory for tmux plugins and related data.
