# home

Top-level dotfiles that are intended to be symlinked directly into the user's home directory (`~`). These files serve as the primary entry points for shell configurations.

## Contents
- `.bashrc`: The main initialization file for interactive Bash shells. It typically sources the modular configurations found in the `bashrc` component.
- `.zshrc`: The main initialization file for interactive Zsh shells. It typically sources the modular configurations found in the `zshrc` component.
