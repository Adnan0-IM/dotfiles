# hypr

Configuration for [Hyprland](https://hyprland.org/), a dynamic tiling Wayland compositor that doesn't sacrifice on looks.

## Contents
- `.config/hypr/`:
    - `hyprland.conf`: The main configuration file defining window rules, monitor setups, and general behavior.
    - `dms/`: Modular configuration directory:
        - `binds.conf`: Keybindings for window management and application launching.
        - `colors.conf`: Color definitions and theme integration.
        - `cursor.conf`: Cursor theme and size settings.
        - `layout.conf`: Tiling layout and window behavior settings.
        - `outputs.conf`: Monitor and display configuration.
        - `windowrules.conf`: Specific rules for individual applications or window types.
