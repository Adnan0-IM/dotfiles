# nvim

Personal Neovim configuration, built on top of [LazyVim](https://www.lazyvim.org/). This setup is designed for a fast, modern, and extensible editing experience.

## Contents
- `.config/nvim/`:
    - `init.lua`: The main entry point for the configuration.
    - `lua/`: Directory containing modular Lua configuration files.
    - `lazy-lock.json`: Lockfile for plugin versions, managed by [lazy.nvim](https://github.com/folke/lazy.nvim).
    - `lazyvim.json`: Configuration for the LazyVim framework.
    - `stylua.toml`: Configuration for the StyLua formatter.
    - `example.lua`: Example or template configuration file.
