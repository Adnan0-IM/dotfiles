# niri

Configuration for [niri](https://github.com/YaLTeR/niri), a scrollable-tiling Wayland compositor.

## Contents
- `.config/niri/`:
    - `config.kdl`: Main configuration file in KDL format, defining layouts, keybindings, and window rules.
    - `dms/`: Modular configuration directory:
        - `binds.kdl`: Keybindings.
        - `colors.kdl`: Color scheme settings.
        - `cursor.kdl`: Cursor configuration.
        - `layout.kdl`: Layout-specific settings.
        - `outputs.kdl`: Monitor and output configuration.
        - `windowrules.kdl`: Window-specific rules.
        - `alttab.kdl`: Alt-tab behavior.
        - `wpblur.kdl`: Wallpaper blur settings.
