# yazi

Configuration for [Yazi](https://github.com/sxyazi/yazi), a blazing fast terminal file manager written in Rust, based on async I/O.

## Contents
- `.config/yazi/`:
    - `yazi.toml`: General configuration and behavior settings.
    - `keymap.toml`: Custom keybindings for navigation and operations.
    - `theme.toml`: Appearance and color scheme configuration.
    - `init.lua`: Initialization script for custom Lua-based functionality.
    - `vfs.toml`: Virtual File System settings.
    - `package.toml`: Package management for Yazi plugins.
    - `plugins/`: Directory for custom or downloaded Yazi plugins.
    - `flavors/`: Directory for color scheme "flavors".
