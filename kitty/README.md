# kitty

Configuration for [Kitty](https://sw.kovidgoyal.net/kitty/), a fast, feature-rich, GPU-based terminal emulator.

## Contents
- `.config/kitty/`:
    - `kitty.conf`: Main configuration file for fonts, window padding, and terminal behavior.
    - `dank-theme.conf`: A custom color theme.
    - `dark-theme.auto.conf`: Automatically managed dark theme settings.
    - `dank-tabs.conf`: Tab bar styling and behavior.
